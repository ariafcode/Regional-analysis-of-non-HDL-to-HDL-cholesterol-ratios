# Data Access & Setup

1. **MainDataset**: Download the Cholesterol dataset directly from the Non-Communicable Disease Risk Factor Collaboration (NCD-RisC) data downloads (https://ncdrisc.org/index.html).
2. **RegionalMap**: The 'ncd_region_map.csv' file is included in this repository
3. **Storage**: Save both files to your Google Drive before executing the notebook.

# Note on Regional Categorization
The country-to-region groupings in 'ncd_region_map.csv' are strictly based on the official NCD-RisC inclusion/exclusion criteria, which organizes individual nations into nine global super-regions.
